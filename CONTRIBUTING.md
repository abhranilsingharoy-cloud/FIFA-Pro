# Contributing to FIFA World Cup 2026 Hub ??

We love your input! We want to make contributing to this project as easy and transparent as possible.

## Pull Requests
1. Fork the repo and create your branch from main.
2. Ensure you use Vanilla CSS / CSS Modules for styling (no heavy frameworks unless discussed).
3. Ensure the project builds successfully (
pm run build).
4. Issue that pull request!

## Bug Reports
Please provide a clear description of the issue and what environment you are running on.
